# React Weather

### About
This is a sample project using React.js and OpenWeatherMap API.

[![](./images/MainPage.png)](#)

### Instructions
1. Clone Repo (if required)
2. Add REACT_APP_WEATHER_API_KEY as an evironment variable
3. Run `npm start`
4. Open [localhost:3000](https://localhost:3000)
